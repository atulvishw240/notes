
>FIFO - First in, First Out

| Operation | Big-O Time |
| --------- | ---------- |
| Enqueue   | O (1)      |
| Dequeue   | O (1)      |
**Enqueue** : Its like pushing to the stack

**Dequeue** : Like popping from the stack but we pop the first element that was inserted. That's why its called FIFO, the element which was inserted first will be popped first too.

 >We can implement queues with arrays, but it will be much more complicated.
