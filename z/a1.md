## A, An or The

**Rule 1:**
A/An => one of many
The => the only one

Ex: I want **an** apple. (from many apples)
I want **the** apple. (the only apple)

**Rule 2:**
A/An => You mention something for the first time
The => You mention something more than once

Ex: I bought **a** T-shirt. **The** (you know which T-shirt you're talking about) T-shirt is lovely. 
Can I open **a** window? Please, close **the** (specific one, which is open) window. 

**Rule 3:**
The => What you speak about is only one of its kind

Ex: Go to **the** red car. (green car, red car, blue car)

## Plurals

**Rule 1**
Singular (1/a) => car
Plural (2 or more) => car + s

**Rule 2**
Singular (1/a) => ends in s, sh or x
Plural (2 or more) => add 'es'
Example : bus => buses
bush => bushes
box => boxes

**Rule 3**
Singular : cons + y
Plural : -ies
Example : baby => babies
dictionary => dictionaries

**Rule 4**
Singular : vowel + y
Plural : -ys
Example : boy => boys

**Rule 4**
Singular : -f, -fe
Plural : -ves
Example : scarf => scarves
knife => knives

**Exceptions**
man => men
child => children
woman => women
foot => feet
tooth => teeth
mouse => mice
sheep => sheep
fish => fish

## A, An or Some

**Rule 1**
A/an => countable noun (singular)
Some => uncountable noun
Example: **a glass of** water (specify quantity)
**some** water (no quantity specified)

>In English **money** is uncountable, that's why we say "Money is important" and not "Money are important"

**Rule 2**
A/an => countable, singular
Some => countable, plural
Example : a chair, some chairs

## Some or Any

Some => countable, plural
